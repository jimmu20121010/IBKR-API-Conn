# Contributing to IBKR-API-Conn

First off, thank you for considering contributing to `IBKR-API-Conn`! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

Here are some guidelines to help you get started with contributing to this Interactive Brokers API integration project.

## How Can I Contribute?

### Reporting Bugs
*   Check the [Issues](https://github.com/jimmu20121010/IBKR-API-Conn/issues) tab to see if the bug has already been reported.
*   If not, open a new issue. Clearly describe the problem, include steps to reproduce, and provide error logs or screenshots if applicable.
*   **Important:** Never include your actual IBKR account numbers, passwords, or live API keys in the logs or issues.

### Suggesting Enhancements
*   Open an issue to discuss your ideas before spending time writing code.
*   Clearly explain the use case and why this enhancement would benefit other users trading with IBKR.

### Code Contributions (Pull Requests)
1.  Fork the repository and create your branch from `main`.
2.  If you've added code that should be tested, please add tests.
3.  Ensure your code follows consistent formatting and clean code practices.
4.  Make sure the API connection handling is secure and doesn't leak sensitive data.
5.  Submit a Pull Request (PR) with a clear description of your changes.

## Code of Conduct

By participating in this project, you agree to abide by basic open-source etiquette: treat others with respect, be welcoming to newcomers, and keep discussions constructive and focused on improving the tool.

## Questions?

If you have questions about using the IBKR API or configuring this connector, feel free to open a discussion or an issue!
